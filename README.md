### ⚡ Clock-Gated Sequential Architecture *(Low-Power Design)*
> Reducing dynamic power consumption at the register level

- Designed **clock-gated flip-flop architecture** to suppress unnecessary switching
- Implemented **clock enable logic** for idle-state power savings
- Compared gated vs non-gated designs using **synthesis and timing analysis reports**
- Studied power-delay trade-offs and their impact on critical timing paths
